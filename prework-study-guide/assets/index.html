HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Closing tags begin with a forward slash
  • div and span elements are non-semantic
  • img elements contain information about images that are displayed on the webpage

  • br element creates a line of empty space, or a line break, between two blocks of content

CSS

  • Enter your CSS notes here
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • internal style sheet contains CSS rules for the webpage in the head section of the HTML file
  • you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code
  • external CSS style sheet allows developers to keep all their CSS rules in a separate file
  • When using an external style sheet, we link the file using a "lin"k element placed in the HTML's "head"
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • the wild card symbol is for when we want to apply a rule to all the elements visible on the page
  • The header selector is for the top margin
  • The footer selector is for the bottom margin
  • we can also apply CSS rules to images
  • a selector defines the element or attributes to which the rules
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.

JavaScript

  • Enter your JavaScript notes here
  • A variable is a named container that allows us to store data in our code.
  • const keyword allows us to create variables that can't be reassigned a value
  • After the const keyword, we give the variable a name
  • we use an assignment operator = to assign the specific data that we want stored in our variable
  • console.log method outputs a message to the web console by adding an argument
  • You link the JS file to the HTML file with in thee body of the HTML
  • Control flow is the order in which a computer executes code in a script.
  • The console is where you will see error messages related to the code
  • if statement is a way of introducing decision-making into our code. Code will only execute if the statement meets a condition
  • we can have more than one condition by using else if
  • An array is a single variable that is used to hold a group of data.
  • To create an array, you declare a variable in the same way that we declared a variable with only one data item, instead of assigning the variable to a single value, we assign the variable to a group of values.
  • arrays are zero-indexed and sequential
  • indices, or location identifiers, will always be whole numbers that increase by 1 each time.
  • A for loop excutes specific task until a specified amount of times. at the same time variable tracking the iterations change as well
  • We can find the length of an array by using a built-in property of an array, appropriately named length
  • a function is a set of instructions that tells the computer how to perform a certain task.
  • To create a function, we use the function keyword. Then we give our function a unique name
  • To call a function you must type NameOfTheFunction() along with any necissary argument within the parentheses
